# TUI Keybinding Conventions (Shared)

Goals:
- Predictable across apps
- Minimal conflicts
- Single-hand friendly

Global (always works):
- q: quit
- ?: help
- r: refresh

Navigation:
- j/k or arrows: move selection
- PgUp/PgDn: fast scroll (focused panel)

Panels:
- 1..9: focus panel or switch tab (simple apps)
- Tab: toggle secondary view in the focused panel

Actions (common):
- a: add
- b: branch or browse
- c: commit or confirm
- o: open in editor
- p: pull
- P: push
- f: fetch
- s: settings

Modal rules:
- Enter: confirm
- Esc: cancel
- Type to filter when list is long
